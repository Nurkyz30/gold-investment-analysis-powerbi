# gold-investment-analysis-powerbi
Анализ инвестиций в золото: Кыргызстан 2020-2025
Интерактивный дашборд в Power BI для анализа динамики цен на золото в Кыргызстане и сравнения доходности альтернативных инвестиционных стратегий.
Что выгоднее за 5 лет - держать деньги на банковском депозите или инвестировать в золото?

Цель проекта:
- Сравнить доходность инвестиций в золото vs банковские депозиты
- Проанализировать динамику цен на золото в период 2020-2025
- Выявить корреляции с курсом доллара США
- Оценить волатильность и риски инвестиций
